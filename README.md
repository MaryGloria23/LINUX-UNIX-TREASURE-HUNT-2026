# Overview

This repository contains the materials for the Linux Scavenger Hunt; a hands-on introductory exercise for the ABI Summer School 2026. Participants download a deliberately messy bioinformatics project directory and work through it using Linux commands, following clues left behind by a fictional researcher, Dr. Kizza, who left the lab in a hurry.
The exercise is designed to be engaging and story-driven. 

It is not a lecture; it is an exploration.
